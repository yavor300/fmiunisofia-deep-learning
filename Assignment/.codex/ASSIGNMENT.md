# 3. Semantic Image Segmentation

## Example

- **Goal:** Create a web application for semantic segmentation of an image of a car driving on a street.
- **Resources:** <https://en.wikipedia.org/wiki/Image_segmentation>
- **Dataset:** Cityscapes dataset — <https://www.cityscapes-dataset.com/dataset-overview/>

## Required Steps

1. Study scientific papers and techniques for semantic image segmentation. Study and compare the loss functions **pixel-wise cross-entropy loss**, **Dice loss**, and **focal loss**.
2. Explore the data: number of observations, number and type of features, statistical relationships in the data, and data anomalies. Prepare a report and analysis illustrated with appropriate visualizations and accompanied by comments.
3. Conduct experiments with an **autoencoder-type neural network**.
4. Demonstrate techniques for changing the learning rate as the number of epochs increases, such as **cosine annealing** and **step decay**.
5. Conduct experiments with different input data preprocessing approaches.
6. Create a user interface using the **Streamlit** package. Prepare tests written in the manner shown during the practical exercises.
7. Prepare a presentation describing the project implementation approach.
